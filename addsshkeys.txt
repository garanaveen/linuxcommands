How to check what keys are added?
Linux,
ssh-add -l -E md5

Convention for the name of the key added.
machinename - ubuntu
workorpersonal - work/personal

Examples:
mac-personal
ubuntu-roku


-------------------------------------------------
Adding ssh keys for github/gitlab, using new ed25519 protocol.
-------------------------------------------------
https://docs.github.com/en/github/authenticating-to-github/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent
ssh-keygen -t ed25519 -C "machinename-workorpersonal"
eval "$(ssh-agent -s)"
Linux,
ssh-add ~/.ssh/id_ed25519-machinename-workorpersonal
Mac,
ssh-add --apple-use-keychain ~/.ssh/id_ed25519-machinename-workorpersonal


Add ~/.ssh/id_ed25519.pub to the github/lab.com settings.
-------------------------------------------------
